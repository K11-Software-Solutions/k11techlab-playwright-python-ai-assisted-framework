# Playwright Python Examples

This directory contains example tests demonstrating Playwright features from basic to advanced.

## Example List
- test_basic_navigation.py: Basic navigation, assertions, and UI interactions
- test_file_upload.py: File upload automation
- test_network_mocking.py: Network request mocking
- test_visual_check.py: Screenshot and visual check
- test_parallel.py: Parallel navigation with pytest parametrize

## Demo URLs
- [Playwright TodoMVC Demo](https://demo.playwright.dev/todomvc)
- [The Internet Herokuapp](https://the-internet.herokuapp.com)
- [SauceDemo](https://www.saucedemo.com)

## Getting Started
Run any test with:
```
pytest tests/playwright-examples --maxfail=1 -v
```

For more, see the [Playwright Python docs](https://playwright.dev/python/docs/intro)
